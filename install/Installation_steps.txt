1. Flash the Raspberry Pi OS (64-bit) Bookworm

2. Update the system:
a) sudo apt update
b) sudo apt upgrade -y
c) sudo reboot

3. Install the libraries:
a) sudo apt-get install libopencv-dev
b) sudo apt-get install python3-opencv
c) sudo apt install python3-matplotlib

4. Prepare the environment; From the terminal:
a) mkdir pi_monte_carlo
b) cd pi_monte_carlo
c) python3 -m venv .virtualenvs --system-site-packages
d) source .virtualenvs/bin/activate
e) gitclone https://github.com/AndreaFavero71/pi_monte_carlo

5. Move below file from pi_monte_carlo/setup folder to the Desktop:
a) pi_start.desktop

6. Give executable property to the start.sh file; From the terminal:
a) cd pi_monte_carlo
b) chmod +x /home/pi/pi_monte_carlo/start.sh

7. Remove the confirmation request when double clicking executable files.
a) Open the File Manager
b) Select Edit
c) Select Preferences
d) At the General tab
e) Check "Don't ask option on launch executable file"

8. Organize the code upload at the power on:
a) sudo crontab –e   (and add below strings at file end:)
MAILTO=""
@reboot su - pi -c "/usr/bin/vncserver-virtual :1 -randr=1280x720"
b) save the file (Ctrl X, Y, Enter)
c) sudo reboot


For the touchscreen display, follow the instruction of the purchased one. In my case, the instructions were:
1. sudo rm -rf LCD-show
2. git clone https://github.com/goodtft/LCD-git
3. chmod -R 755 LCD-show
4. cd LCD-show/
5. sudo ./LCD7C-show (this commands quickly plots a couple of commands and the board reboot)
